## 🚨 Overview

**DAO Treasury Risk Guardian** is a full-stack Web3 monitoring system that tracks treasury wallets in real time and detects risk events such as:

- 🔻 Sudden balance drops  
- 🧠 Suspicious activity  
- ⚠️ Risk anomalies  

Built for:
- DAOs  
- Protocols  
- Treasury managers  
- Security teams  

---

## 🎯 Problem

Most DAOs:

- Do NOT monitor treasury wallets in real-time  
- Detect issues too late  
- Lack automated risk alerting  

👉 Result: Loss of funds, delayed response, poor visibility

---

## 💡 Solution

This system provides:

- 🔄 Continuous wallet tracking  
- 📊 Risk analysis engine  
- 🚨 Instant alert generation  
- 📡 Real-time dashboard visibility

## ⚙️ Tech Stack

### Backend
- Node.js + TypeScript  
- Express  
- MongoDB (Atlas)  
- JWT Authentication  

### Workers
- Tracker Worker (Blockchain monitoring)  
- Risk Engine Worker (alert generation)  

### Frontend
- React (Vite)  
- Context API  
- Axios  

### Web3
- Alchemy RPC  
- Ethereum (Sepolia / Mainnet ready)
